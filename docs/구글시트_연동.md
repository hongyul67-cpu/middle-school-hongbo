# 구글시트 실시간 연동 (설치 없이 팀 공유)

웹 링크는 그대로 두고, 기록을 **공용 구글시트**에 저장해 여러 선생님이 실시간으로 함께 씁니다.
**관리자가 딱 한 번**(약 5분) 설정하면, 이후엔 누구나 설치 없이 링크만으로 사용해요.

---

## A. 관리자 1회 설정

1. **새 구글 스프레드시트**를 만듭니다. (이름 예: `홍보 배차판 기록`)
2. 상단 메뉴 **확장 프로그램 → Apps Script**.
3. 편집기에 있던 내용을 지우고, 저장소의 **`integrations/google-sheets/Code.gs`** 내용을 통째로 붙여넣고 💾 저장.
4. 오른쪽 위 **배포 → 새 배포**.
   - 톱니바퀴(유형 선택) → **웹 앱**
   - 설명: 아무거나 (예: v1)
   - **다음 사용자 인증 정보로 실행: 나**
   - **액세스 권한이 있는 사용자: 모든 사용자**
   - **배포** 클릭 → 권한 요청이 뜨면 본인 구글 계정으로 승인.
5. 나오는 **웹 앱 URL**을 복사합니다. (`https://script.google.com/macros/s/····/exec` 형태)

> 처음 배포 시 "확인되지 않은 앱" 경고가 나올 수 있어요. **고급 → (안전하지 않음으로) 이동**을 눌러 본인 계정으로 승인하면 됩니다. 본인이 만든 스크립트라 안전합니다.

---

## B. 앱에 연결

**방법 1 — 앱에서 붙여넣기**
1. 웹 앱을 엽니다: `https://hongyul67-cpu.github.io/middle-school-hongbo/`
2. 우측 상단 **연결**(점 배지) 클릭 → 복사한 `/exec` URL 붙여넣기 → **저장하고 연결**.
3. 배지가 **● 구글시트**로 바뀌면 성공.

**방법 2 — 링크 하나로 자동 연결(추천, 공유용)**
- 주소 뒤에 `?api=`+URL 을 붙여 공유하면 받는 분은 붙여넣기 없이 바로 연결됩니다:
  ```
  https://hongyul67-cpu.github.io/middle-school-hongbo/?api=여기에_exec_URL
  ```
- 이 링크를 단톡방에 공유하거나 휴대폰 홈화면에 추가하세요.

---

## 어떻게 저장되나

- 시트의 **`기록`** 탭에 학교별로 한 줄씩 쌓입니다 — 학교명·지역·방문상태·1~6차 날짜·방문자·관심학생수·반응·다음할일·비고.
- 맨 끝 `_t` 열은 **병합용 시간표시**(숨김)라 건드리지 않아도 됩니다.
- 여러 명이 **같은 학교의 다른 항목**을 동시에 고쳐도 서로 덮어쓰지 않게 필드별로 병합돼요.
- 앱은 15초마다 시트를 다시 읽어 팀원 변경을 자동 반영합니다.

## 자주 묻는 것

- **연결이 안 돼요(연결 끊김).** → ① URL이 `/exec`로 끝나는지 ② 배포 시 "액세스: 모든 사용자"로 했는지 확인. 수정 후 **새 배포**(또는 배포 관리에서 버전 갱신)하면 됩니다.
- **인터넷이 끊기면?** → 그 사이 입력은 이 기기에 임시 저장되고, 연결이 돌아오면 다시 반영을 시도합니다. (중요한 변경은 연결 상태(● 구글시트)를 확인하세요.)
- **시트를 직접 고쳐도 되나요?** → 앱을 통해 쓰는 걸 권장합니다. 시트에서 직접 크게 손봤다면, 앱의 **가져오기(CSV)** 로 맞추는 방법도 있어요.
- **개인정보** → 방문자는 우리 담당자명(내부 운영용)만. 대상 중학교의 교사·학생 정보는 넣지 마세요.
