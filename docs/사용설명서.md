# 홍보 배차판 — 사용설명서 (처음 쓰는 분용)

어렵지 않아요. **학교 고르고 → 다녀와서 적기**가 전부입니다.

---

## 1. 열기 (셋 중 편한 것)

- **웹으로 바로**: 아래 주소를 누르면 열립니다. 휴대폰 홈화면에 추가해두면 앱처럼 써요.
  `https://hongyul67-cpu.github.io/middle-school-hongbo/`
- **파일로**: `dist/홍보배차판_단독실행.html` 파일을 받아 더블클릭 (인터넷 없어도 됨).
- 처음 열면 **간단 사용법 안내**가 한 번 뜹니다. 다시 보려면 오른쪽 위 **?** 를 누르세요.

---

## 2. 화면 보는 법

- **맨 위 숫자**: 대상 학교 수 · 방문완료 · 설명회 · 관심학생 합계 · 진행률.
- **목록**: 위에서부터 **먼저 갈 학교**예요. (우리 학교로 많이 진학한 순서로 자동 정렬)
- 학교 옆 **상 / 중 / 하**: 우선순위 등급. 왼쪽 숫자는 점수.
- **학과 미배정**: 담당 학과가 아직 안 정해진 학교라는 표시입니다.

---

## 3. 방문 기록하기 (핵심)

1. 학교 카드를 **한 번 누르면** 아래로 펼쳐져요.
2. **방문 상태**를 고르세요 — 미방문 / 방문완료 / 재방문필요 / 설명회예정 / 관심없음.
3. **방문 회차**: 다녀온 날 회차의 **`오늘`** 버튼을 누르면 날짜가 자동으로 들어갑니다.
   (다른 날짜는 달력에서 직접 선택. 잘못 넣었으면 **✕**로 지워요.)
4. **관심 학생 수 · 반응 · 다음 할 일 · 메모**를 적으세요.
5. 적는 즉시 **자동 저장** — 따로 저장 버튼 없어요.

---

## 4. 찾기 · 좁혀 보기

- **학교명 검색**: 맨 위 칸에 학교 이름 입력.
- **필터**: 지역 / 방문상태 / 등급(상·중·하) / **학과 미배정만** 을 눌러 좁혀 봐요.
- **정렬**: 우선순위 · 지역 · 상태 중 선택.
- **지도**: 카드를 펼치면 **네이버지도 · 카카오맵** 버튼으로 위치가 열립니다.

---

## 5. 같이 쓰기 · 자료 모으기

- 웹/앱 버전은 기록이 자동 저장돼요. (설치형 팀 공유판은 **15초마다** 서로의 입력이 자동 반영됩니다.)
- **내보내기**: 맨 위 `내보내기` → 엑셀로 열 수 있는 CSV 파일이 저장돼요. 공용 시트에 붙여넣기 좋아요.
- **가져오기**: 공용 시트에서 정리한 CSV를 `가져오기` 로 불러오면 방문기록이 합쳐집니다.

---

## 자주 묻는 것

- **저장 버튼이 없어요?** → 적는 순간 자동 저장이라 없어도 됩니다.
- **웹 버전인데 다른 선생님 기록이 안 보여요.** → 웹/파일 버전은 그 기기에만 저장돼요. 실시간 공유가 필요하면 팀 공유판(설치형)을 쓰거나, `내보내기/가져오기`로 자료를 모으세요.
- **학교가 두 번 보여요.** → 중복은 정리했어요. 그래도 이상하면 알려주세요.

> 개인정보(교사·학생 이름·연락처)는 이 도구에서 다루지 않습니다. 학교 단위 정보만 기록하세요.
