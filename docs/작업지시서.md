# 작업지시서 — 2027 신입생 모집 홍보 관리 도구

> 대상: 경기기계공업고 취업지원부(홍보부)
> 산출물: ① `홍보배차판.jsx` (React 현장 앱), ② `홍보대상학교_마스터.xlsx` (공유·편집 원장)
> 원칙: **개인정보(교사·학생 이름·연락처) 미포함.** 학교 단위 데이터만 다룬다.

---

## 1. 개요

과거 우리 학교로 진학한 실적을 기준으로 **홍보 대상 중학교의 우선순위**를 매기고,
출장 방문(1~6차)과 결과를 기록·공유하는 도구. 총 **144개교**.

- 데이터 출처: `2027_신입생_모집_홍보_학교_배정.xlsx` (사용자 업로드 원본)
- 방문 회차 구조는 원본 `홍보부만_공유_시트.xlsx`의 **복명서 시트(1~6차)**에서 차용
- 관심학생·상담 명단 등 **개인정보 포함 시트는 이 도구에 통합하지 않음** → 도구 밖에서 별도 관리

---

## 2. 공통 규칙

### 2-1. 우선순위 점수 (양쪽 동일)
```
점수 = (최근3년 입학생수 × 2) + (2026 입학생수 × 3) + (3학년 학급수 × 0.6)
```
- 값이 비면 0으로 취급.
- 등급 경계(2027 데이터 상위 33% / 66% 기준): **상 ≥ 17.0**, **중 ≥ 8.6**, 그 미만 **하**.
- 공식·경계값을 바꾸면 양쪽(앱 상수 / 시트 '안내'탭 B7·B8)을 **함께** 수정할 것.

### 2-2. 학교 데이터 스키마
| 키(앱) | 시트 컬럼 | 의미 | 비고 |
|---|---|---|---|
| `n` | 학교명 | 학교명 | |
| `g` | 지역 | 구/시 | 예: 노원구, 남양주시 |
| `s` | 시도 | 서울시/경기도 | |
| `y3` | 최근3년입학생수 | 3년 누적 진학 실적 | 핵심 지표 |
| `y26` | 2026입학생수 | 최근년도 실적 | |
| `c3` | 3학년학급수 | 규모(잠재 모집수) | |
| `st` | 학생수 | 재학생 수 | 결측 많음 |
| `d` | 대표학과 | 담당 학과 | **76개교 미배정(공백)** |
| `sc` | 우선순위점수 | 계산값 | |
| `t` | 등급 | 상/중/하 | |

### 2-3. 방문 기록 필드 (사용자 입력)
`방문상태` · `1~6차(날짜)` · `관심학생수` · `반응` · `다음할일` · `메모`
- 방문상태: `미방문 / 방문완료 / 재방문필요 / 설명회예정 / 관심없음`
- 반응: `좋음 / 보통 / 미온 / 부재중`

## 3. ① 홍보배차판.jsx (React 앱)

### 기술
- 단일 파일 React, `export default function App`. 외부 라이브러리 없음(순수 React + 인라인 스타일).
- 폰트: 데이터·숫자는 모노스페이스(`MONO`), 본문은 시스템 산세리프(`SANS`).
- 디자인 토큰: 상단 `C` 객체(잉크 네이비 `#14243B` + 신호 앰버 `#E0912B`). 색 변경은 여기서.

### 상태 / 저장
- 학교 목록은 `SCHOOLS` 상수로 **하드코딩 임베드**(144개).
- 방문 기록은 `window.storage` 키 `"outreach-state-v1"`, **`shared: true`(팀 공유)**.
  - 구조: `{ [학교명]: { status, rounds:[6], interest, reaction, next, memo } }`
  - 저장 함수 `mergeSave()`가 매 수정마다 get→merge→set (전체 객체 저장).
- CSV 내보내기: BOM 포함, 시트 컬럼 순서와 동일 → 시트에 붙여넣기 가능.

### 주요 컴포넌트
- `App` — 필터/정렬/통계/목록 총괄
- `Chip` — 지역·상태 필터 칩
- `SchoolCard` — 카드(점수·등급·라운드 막대·상태) + 펼치면 상세 입력 폼
- `Field` — 라벨+입력 래퍼

### 현재 기능
- 지역·상태 필터, 학교명 검색, 정렬(우선순위/지역/상태)
- 상단 요약(대상/방문완료/설명회/관심학생 합계/진행률)
- 회차 막대(1~6차) + 상세에서 회차 누르면 오늘(`M/D`) 기록/삭제
- 네이버 지도 검색 링크(학교명 기준)

## 4. ② 홍보대상학교_마스터.xlsx

- 시트 2개: **안내**(사용법·등급 경계값 B7/B8), **홍보대상학교**(본표, 점수순 정렬).
- 점수는 수식 `=E*2+F*3+G*0.6`, 등급은 `=IF(점수>=안내!$B$7,"상",IF(...,"중","하"))` → 자동 재계산.
- 노란 칸 = 사용자가 채우는 칸. 방문상태·반응은 드롭다운(데이터 검증).
- C2 틀 고정. 담당 교사 컬럼 없음(대표학과로 운영).

## 5. 알려진 이슈 / 백로그

| 우선 | 항목 | 내용 |
|---|---|---|
| 높음 | 대표학과 미배정 | 76개교 공백 → 배정 채우기(양쪽) |
| 높음 | 시트→앱 동기화 | 현재 앱은 CSV **내보내기만**. **가져오기(import)** 없음 → 시트에서 수정한 값이 앱에 반영 안 됨 |
| 중 | 동시편집 충돌 | `shared` 저장이 last-write-wins. 여러 명 동시 입력 시 덮어쓸 수 있음 → 학교별 키 분리 또는 타임스탬프 병합 검토 |
| 중 | 회차 날짜 연도 | 현재 `M/D`만 기록 → 연도 포함 또는 날짜 선택 UI |
| 중 | 지도 정밀도 | 이름 검색 링크 수준. 주소/좌표 추가 시 핀·길찾기 가능(학교알리미에서 주소 보강) |
| 낮음 | 전화번호 | 원본에 없음 → 필요 시 학교알리미에서 보강 |
| 낮음 | 등급 필터·백업/초기화 버튼 | 편의 기능 |

`SCHOOLS` 배열(JSON)로 앱에 주입, 동시에 시트도 동일 데이터로 재빌드하면 양쪽이 항상 일치한다.
