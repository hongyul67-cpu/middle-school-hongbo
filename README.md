# 신입생 모집 홍보 관리 도구

경기기계공업고 취업지원부(홍보부)가 **홍보 대상 중학교를 우선순위로 정리하고, 방문(1~6차)을 다니며 결과를 기록·공유**하기 위한 도구입니다. 홍보물(포스터·전단)이 아니라, **홍보 다니기를 돕고 다녀와서 자료를 수합**하는 업무 도구예요.

> **원칙: 개인정보(교사·학생 이름·연락처)는 다루지 않습니다.** 학교 단위 데이터만 씁니다.

---

## 무엇이 들어있나

| 경로 | 내용 |
|---|---|
| **`app/홍보배차판.jsx`** | 현장에서 쓰는 React 앱. 학교 목록·우선순위·방문기록·CSV 가져오기/내보내기 |
| **`data/홍보대상학교_마스터.xlsx`** | 공유·편집용 원장(안내 시트 + 본표, 점수·등급 자동 수식) |
| **`data/대표학과_미배정_76개교_배정용.csv`** | 대표학과가 빈 76개교를 채우기 위한 입력지(지역별 기배정 학과 힌트 포함) |
| **`data/학교_주소좌표_입력용.csv`** | 지도 핀·길찾기를 위한 주소/좌표 입력지 |
| **`scripts/rebuild_data.py`** | 위 입력지를 병합해 앱 `SCHOOLS`와 마스터 엑셀을 **동시 갱신**, 점수·등급 재계산 |
| **`scripts/geocode.py`** | (네트워크 필요) 좌표 자동 채우기(OpenStreetMap, 베스트에포트) |
| **`docs/작업지시서.md`** | 전체 규칙·데이터 스키마·백로그 원문 |

현재 대상 **144개교** — 등급 상 48 · 중 51 · 하 45 / 서울시 81 · 경기도 63 / 18개 지역.

---

## 우선순위 점수

과거 우리 학교로 진학한 실적을 기준으로 대상 학교의 우선순위를 매깁니다.

```
점수 = (최근3년 입학생수 × 2) + (2026 입학생수 × 3) + (3학년 학급수 × 0.6)
```

| 등급 | 경계값 |
|---|---|
| **상** | 점수 ≥ 17.0 |
| **중** | 점수 ≥ 8.6 |
| **하** | 그 미만 |

공식·경계값을 바꾸면 **앱 상수**와 **시트 '안내'탭 B7/B8**, `scripts/rebuild_data.py`의 `TIER_HIGH/TIER_MID`를 **함께** 수정하세요.

## 쓰는 방법

### 1) 앱(홍보배차판)
- 지역·상태 필터, 학교명 검색, 정렬(우선순위/지역/상태)
- 카드를 펼쳐 **방문상태 · 회차(날짜 선택) · 관심학생수 · 반응 · 다음할일 · 메모** 입력
- 네이버지도·카카오맵 열기(좌표가 있으면 길찾기까지)
- 기록은 팀 공유 저장소에 저장되고, **15초마다 팀원 변경이 자동 반영**됩니다

### 2) 시트 ↔ 앱 순환
- 앱 **내보내기** → 시트 컬럼과 같은 순서의 CSV 저장 → 시트에 붙여넣기
- 시트에서 정리·수정한 CSV를 앱 **가져오기** → 방문기록이 병합(학교명+지역 기준)

### 3) 데이터 채우기(대표학과·주소·좌표)
```bash
# 1. data/*_입력용.csv 를 채운다 (엑셀/구글시트로 열어 작성)
# 2. 미리보기
python3 scripts/rebuild_data.py
# 3. 앱 SCHOOLS + 마스터 엑셀에 반영
python3 scripts/rebuild_data.py --write
```
좌표를 자동으로 시도하려면(외부 네트워크 필요):
```bash
python3 scripts/geocode.py --limit 10   # 테스트
python3 scripts/geocode.py               # 전체
python3 scripts/rebuild_data.py --write  # 검수 후 반영
```

## 앱을 개발용으로 열어보려면

단일 파일 React 컴포넌트(`export default function App`)입니다. 외부 라이브러리 없이 순수 React로 동작하며, 팀 공유 저장을 위해 `window.storage`(get/set, shared) API를 사용합니다. 문법 확인은:

```bash
npx esbuild app/홍보배차판.jsx --loader:.jsx=jsx --bundle --external:react --format=esm --outfile=/dev/null
```
